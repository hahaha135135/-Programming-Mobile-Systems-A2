Help & Support

Frequently Asked Questions (FAQ)

Q: How do I add a new item to the inventory?
A: Go to the Inventory page, fill in all required fields in the form (ID, Name, Category, Quantity, Price, Supplier, Stock Status, Popular), then click the "Add Item" button. The item will appear in the table below.
Q: How do I edit an existing item?
A: On the Inventory page, locate the item you want to edit in the table and click the "Edit" button. The form will be populated with the item's data. Make your changes and click "Update Item".
Q: How do I delete an item?
A: On the Inventory page, find the item in the table and click the "Delete" button. A confirmation dialog will appear - click OK to confirm deletion.
Q: Can I search for items?
A: Yes. Go to the Search page, enter an item name (or partial name) in the search box, and click "Search". You can also click "Show Popular Items" to view only popular items.
Q: Is my data saved after I close the browser?
A: No. This application stores data only in memory during your current session. All data will be lost when you close the browser tab. This is by design for this assessment.
Q: What does "Popular Item" mean?
A: "Popular Item" is a flag you can set when adding or editing an item. It indicates items that are in high demand or best-sellers. You can filter to view only popular items on the Search page.

Troubleshooting Guide

Problem: Item ID already exists error
Solution: Each item must have a unique ID. Check the existing items in the table and use a different ID number.
Problem: Item name already exists error
Solution: Item names must be unique. Either use a different name, or edit the existing item instead of adding a new one.
Problem: Quantity or Price shows as invalid
Solution: Quantity and Price must be non-negative numbers. Make sure you are not entering negative values or letters.
Problem: Cannot find an item when searching
Solution: Search is case-insensitive and matches partial names. Try entering just part of the item name. Click "Reset" to see all items.
Problem: Table is not displaying properly on mobile
Solution: The application is responsive. On small screens, you can scroll horizontally to see all columns. Rotate your device to landscape for better viewing.
Problem: Changes not showing after update
Solution: After clicking "Update Item", the table should refresh automatically. If not, try navigating to another page and back, or refresh the browser.

Getting Additional Support

If you continue to experience issues:

  • Check the browser console for errors (F12 > Console tab)
  • Make sure you are using a modern browser (Chrome, Firefox, Edge, Safari)
  • Clear your browser cache and reload the page
